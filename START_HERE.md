## 🚀 Getting Started (3 Steps)

### Step 1: Open the Platform

**Option A: Simple (Double-click)**
1. Navigate to the `torah-platform` folder
2. Double-click `index.html`
3. Your browser will open the homepage

**Option B: Local Server (Recommended for best performance)**

If you have Python installed:
```bash
cd torah-platform
python -m http.server 8000
```

Then open: `http://localhost:8000`

If you have Node.js installed:
```bash
cd torah-platform
npx http-server
```

Then open the URL shown in terminal.

## 🆘 Troubleshooting

### Common Issues

**Hebrew text not displaying?**
- Ensure fonts are loading (check browser console)
- Try refreshing the page
- Use modern browser

**Parsha not loading?**
- Check internet connection (uses Hebcal & Sefaria APIs)
- Wait a moment for API response
- Check browser console for errors

**Styles look broken?**
- Clear browser cache
- Ensure CSS files are loaded
- Use modern browser

**APIs not responding?**
- Hebcal/Sefaria may have rate limits
- Wait a moment and retry
- Check network tab in browser dev tools
